# 🌾 AICAM Pellet Analyzer

เว็บแอปวัดขนาดเม็ดอาหารสัตว์ด้วยกล้องมือถือ — ถ่ายรูป วัดความยาว/เส้นผ่าศูนย์กลาง วิเคราะห์การกระจายขนาดและสี แล้วบันทึกรายงานออนไลน์

🔗 **ใช้งานออนไลน์:** https://supiwit.github.io/aicam-pellet/ (รหัสเริ่มต้น `1234` — เปลี่ยนในหน้าตั้งค่า)

## ฟีเจอร์

- 🔐 ต้องกรอกรหัสก่อนเข้าใช้งาน (ตั้งรหัส 4-8 หลักได้เอง, จดจำเครื่อง 7 วันได้)
- 🌏 รองรับ 3 ภาษา: ไทย / Tiếng Việt / English
- 📷 ถ่ายรูปจากกล้องมือถือโดยตรง หรือเลือกรูปจากเครื่อง
- 📏 คาลิเบรตมาตราส่วนโดยแตะ 2 จุดบนวัตถุอ้างอิง (ไม้บรรทัด เหรียญ ฯลฯ)
- 🔬 ตรวจจับเม็ดอาหารอัตโนมัติ วัด **ความยาว** และ **เส้นผ่าศูนย์กลาง** (มม.) รายเม็ด
  - แก้แสงไม่สม่ำเสมอในภาพอัตโนมัติ (illumination correction)
  - แยกเม็ดที่ติดกันหัวต่อหัวอัตโนมัติ (neck splitting)
  - Ø วัดจาก median ความกว้างช่วงกลางเม็ด แม่นกว่ากรอบสี่เหลี่ยม
- 🎯 เป้าหมายตามไซซ์อาหาร (Die): กำหนดช่วงขนาด + % Insize ขั้นต่ำของแต่ละหัวดาย
  แสดง **Under / Insize / Over** พร้อมตัดสินผ่าน/ไม่ผ่าน (ค่าเริ่มต้น ≥60% ตามตารางโรงงาน)
- 📊 สถิติ: จำนวนเม็ด, ค่าเฉลี่ย, SD, ค่าต่ำสุด/สูงสุด
- 📈 กราฟแท่งความยาว + กราฟโดนัท % สัดส่วน + กราฟการกระจาย Ø
- 🎨 วิเคราะห์สีเฉลี่ยของเม็ด เปรียบเทียบกับสีอ้างอิง (ค่า ΔE) พร้อมเกณฑ์ผ่าน/ไม่ผ่าน
- 💾 บันทึกผลขึ้นฐานข้อมูลออนไลน์ (Supabase) พร้อมรูปที่วิเคราะห์แล้ว — เปิดดูจากเครื่องไหนก็ได้
- 🔗 แชร์รายงานเป็นลิงก์ (`?report=<id>`) ส่งต่อทาง LINE/อีเมลได้ ผู้รับเปิดดูได้ทันที
- ⬇️ ส่งออก CSV ทั้งรายงานเดี่ยวและสรุปทั้งหมด

## วิธีใช้งาน

1. **เปิดแอป** บนมือถือหรือคอมพิวเตอร์ (ดูหัวข้อ "การติดตั้ง" ด้านล่าง)
2. **ถ่ายรูป** — วางเม็ดอาหารบน **พื้นเข้ม** กระจายไม่ให้เม็ดติดกัน วางวัตถุอ้างอิงที่ทราบความยาว (เช่น ไม้บรรทัด) ไว้ในภาพ ถ่ายจากมุมตั้งฉาก
3. **คาลิเบรต** — กดปุ่ม 📏 คาลิเบรต แล้วแตะ 2 จุดที่ปลายวัตถุอ้างอิง กรอกความยาวจริง (มม.)
   - ถ้าใช้ขาตั้งกล้องระยะคงที่ คาลิเบรตครั้งเดียวพอ ค่าจะถูกจำไว้
4. **วิเคราะห์** — กด 🔬 วิเคราะห์ภาพ ระบบจะตีกรอบทุกเม็ดพร้อมแสดงผล
5. **บันทึก** — กรอกชื่อตัวอย่าง/ล็อต แล้วกด 💾 บันทึกขึ้นออนไลน์
6. **ดูรายงาน** — แท็บ 📊 รายงาน แสดงประวัติทั้งหมด แตะเพื่อดูละเอียด/ลบ/ส่งออก CSV

## เคล็ดลับให้วัดแม่น

- ใช้พื้นหลังสีเข้มสนิท (เม็ดอาหารสีอ่อน) — ตั้งค่า "พื้นหลังภาพ" ได้ในแท็บตั้งค่า
- แสงสว่างสม่ำเสมอ ไม่มีเงาเข้ม
- เม็ดต้องไม่ติดกัน — ก้อนที่ใหญ่ผิดปกติหรือชนขอบภาพจะถูกตัดออกอัตโนมัติ (กรอบขาวจาง)
- ถ่ายตั้งฉากกับพื้น (ใช้ขาตั้งจะดีที่สุด)

## การติดตั้ง / เปิดใช้บนมือถือ

แอปเป็นไฟล์ static ล้วน (HTML/CSS/JS) ใช้ได้ 2 วิธี:

**วิธีที่ 1 — เปิดผ่าน LAN (ทดลองใช้เร็ว)**
```bash
cd pellet-app
npx http-server -p 8080
# แล้วเปิดจากมือถือ (Wi-Fi เดียวกัน): http://<IP เครื่องคอม>:8080
```

**วิธีที่ 2 — Hosting ฟรี (ใช้งานจริง เปิดได้ทุกที่)**
อัปโหลดโฟลเดอร์ `pellet-app` ขึ้น Netlify Drop (ลากวางที่ app.netlify.com/drop), GitHub Pages, Cloudflare Pages หรือ Vercel — ได้ URL ถาวรไว้เปิดจากมือถือทุกเครื่อง ข้อมูลทุกเครื่องเชื่อมฐานข้อมูลออนไลน์ตัวเดียวกัน

## โครงสร้างระบบ

| ส่วน | เทคโนโลยี |
|---|---|
| ตรวจจับ/วัดเม็ด | JavaScript ล้วน (Otsu threshold + connected components + PCA) ประมวลผลในเครื่อง |
| กราฟ | Chart.js |
| ฐานข้อมูล + เก็บรูป | Supabase (โปรเจกต์ `aicam-pellet`, ตาราง `measurement_sessions`, bucket `pellet-images`) |

> หมายเหตุ: ฐานข้อมูลเปิดสิทธิ์อ่าน/เขียนสาธารณะผ่าน anon key (เหมาะกับใช้ภายในทีม) หากต้องการจำกัดผู้ใช้ ให้เพิ่มระบบล็อกอิน Supabase Auth ภายหลังได้
